# 드럼 패드 탭 — 아이디어 스케치

> 상태: **아이디어 단계**. 미착수.

## 동기 (사용자 아이디어)

새로운 탭으로 4버튼 드럼 패드를 구현한다. 학생이 태블릿을 쓰고, 전자 드럼 루프의
기초를 설명하려면 필요한 기능이다.

## 핵심 사양 (사용자 아이디어)

- **4개 패드로 강제**: ① kick ② snare ③ closed hi-hat ④ crash cymbal
- `~/data/`의 Cymatics one-shot 샘플 중 몇 개만 골라 **프리셋**으로 만들어 둔다
  - 후보 파일 (data/Cymatics - Dope Drums Vol 1/One Shots/):
    Kick (Knock/Sturdy/Bounce/Bound), Snare (Join/Sniper), Hihat (Check/Traplord),
    Open Hat (House/One), Clap·Rim·Percussion 등

## Claude 제안

### 1. 라이브 탭 + 16-step 시퀀서 그리드 (둘 다)
패드를 직접 두드리는 라이브 모드만으로는 "두드리는 장난감"에서 끝난다.
**16-step 그리드**(1마디 ×16 또는 4마디)를 함께 두면, `harness-music-quality.md`의
드럼 규칙(스네어는 2·4박 등)이 **눈에 보이는 칸 위치**로 학습된다.
- 라이브 모드: 패드 두드려 음색 익히고 즉흥 연주
- 시퀀서 모드: 칸을 토글해 패턴을 "구성" → 규칙을 시각적으로 체득

### 2. 플레이헤드 시각 피드백
루프를 도는 동안 재생 위치 막대가 그리드 위를 지나가게 한다.
"지금 2박" "지금 4박"을 학생이 보면서 들어야 규칙이 와닿는다.

### 3. 객관 모드로 export — dead-end 방지
드럼 패드에서 만든 패턴을 **"이 드럼을 작품에 넣기"** 버튼으로
객관 모드 레이어 작곡기에 `s("...")` 드럼 레이어로 보낸다.
드럼 패드가 독립 장난감이 아니라 메인 작곡 흐름의 입구가 된다.

### 4. 챌린지 모드 (선택)
"스네어를 2박·4박에 놓아보세요" 같은 미션 + 그리드에서 정오 확인.
규칙을 행동으로 연결.

### 5. 저지연 재생
드럼 패드는 탭→소리 지연이 체감을 좌우한다. 프리로드한 `AudioBuffer`를
`AudioBufferSourceNode`로 직접 재생하고 — Strudel 스케줄러를 거치지 않는다.
(Strudel은 객관 모드 작곡기용, 드럼 패드는 자체 경량 재생 경로)

### 6. 키트는 일단 1개 큐레이션
음색 다양성보다 단순함 우선. 잘 어울리는 4종 1키트로 시작하고,
여유 되면 "어쿠스틱풍 / 전자(트랩)풍" 2키트 정도까지만.

## 열린 질문

### Q1. Cymatics 샘플 사용 — ✅ 결정됨: 사용
라이선스(`data/Cymatics - Dope Drums Vol 1/License...pdf`) 요약:
- 로열티 프리, 개인·상업 프로젝트·배포·플랫폼 업로드 OK, 크레딧 불필요
- **clause d: 샘플을 "또 다른 제품으로 / 별도 사운드로" 재배포·재판매 금지**

**근거**: clause d의 취지는 "샘플을 재포장/재판매하지 마라"이다. 이 사용은 —
폐쇄된 수업용 앱 + 전체 팩이 아닌 종류별 소수 샘플(Kick/Snare/Hihat/Cymbal n개씩)
+ 기능적 사용 + 비매(非賣) — 이라 clause d의 취지를 건드리지 않는다.

> ⚠️ 정정: "올린 샘플은 다운로드 불가능" 전제는 기술적으로 성립하지 않는다.
> 브라우저가 오디오를 재생하려면 그 `.wav`를 반드시 내려받으므로, 개발자도구나
> URL 직접 접근으로 원본 회수가 가능하다. 따라서 사용 근거는 "다운로드 불가"가
> 아니라 **"재배포·재판매가 아님"** 이 맞다 — 결론(사용)은 동일.

작은 완화책(필수 아님): 종류별 최소 개수만, 폴더 구조를 "샘플 팩" 형태로 두지 않기,
파일명에서 Cymatics 브랜딩 노출 최소화.

### Q2. 샘플 음색 통일성
드럼 패드(Cymatics)와 하네스 드럼 레이어(`dirt-samples`)의 음색이 다르면
같은 앱 안에서 톤이 따로 논다. Q1과 묶어서 — 양쪽이 같은 큐레이션 키트를
쓰도록 통일할지 검토.

### Q3. 루프 길이 — ✅ 결정됨: 4마디
드럼 패드 그리드도 4마디 기준. crash 규칙("8마디 단위")은
`harness-music-quality.md` Q1 참고 — 4마디 루프에선 단순화해 "루프 첫 박"으로 본다.
